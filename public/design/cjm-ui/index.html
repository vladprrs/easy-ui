EASY-UI · ПРОДУКТОВЫЙ ДИЗАЙН

CJM и интерфейс просмотра кликабельных прототипов

easy-ui — вьюер JSON-прототипов: галерея флоу, кликабельный плеер, карта пути по экранам и библиотека компонентов. Ниже — путь стейкхолдера и hi-fi макеты пяти ключевых экранов.

01

CJM — путь стейкхолдера

Персона: продакт или заказчик, который получил ссылку на прототип. Он не пишет JSON — он кликает флоу, смотрит карту экранов и даёт обратную связь.

02

UI ключевых экранов

Пять экранов по этапам пути: галерея, плеер, CJM-вид, редактор и библиотека компонентов. Плоские плашки, крупные радиусы, лавандовая палитра, Coil для заголовков и YS Text для интерфейсного текста.

Живой Storybook сгруппирован по атомарным уровням. Стейкхолдер видит, из каких «кубиков» собран прототип, и ссылается на конкретный компонент в фидбэке.

Принципы интерфейса

  • 01Одна ссылка — весь контекст: плеер, карта и версия доступны из любой точки
  • 02Кликабельность видима: хотспоты и подсказки убирают «слепые клики»
  • 03CJM — живые рендеры, а не скриншоты: карта не устаревает
